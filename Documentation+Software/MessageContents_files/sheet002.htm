Byte Meaning Notes Message Containing Image Segment
9 Segment Start - Row - High Byte Indicates the row and column for the start of this line of pixels.
10 Segment Start - Row - Low Byte  A segment of a row's columns are sent in a single message.
11 Segment Start - Column - High Byte
12 Segment Start - Column - Low Byte
13 Number of Pixels in Message Total number of pixels in the message. This accounts for a variable number of columns contained in a message.
14 Pixel Depth Pixels have depth. For example: black/white and grayscale have depth 1, RGB has depth 3, NRGB has depth 4.
15 Red Example: RGB for a given pixel.
16 Green
17 Blue
18 other pixels as needed All pixels have the number of paramaters equal to the image depth. Must fit within the maximum message length.
