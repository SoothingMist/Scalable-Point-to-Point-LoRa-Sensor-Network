Byte Meaning Notes Request for sensor reading or actuator operation
7 0..255. Indicates the sensor or actuator identifier. Ex: BATV refers to "battery value", SLED refers to standard onboard LED. See appropriate *.h file.
9,10,11,12 If an actuator, the number of milliseconds of activation (high/low bytes). If zero, turn on if off and leave on until told to turn off. If already on, turn off.
(Leaving on for a given number of milliseconds has not yet been implemented.)
